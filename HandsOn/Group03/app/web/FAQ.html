  3. What are the transformations and mappings performed?

    Transformations and Mappings:

    1. Adapt our ontology to an already existing one from schema.org.
    2. Script to make the YARRRML file using the ttl file.
    3. Cleaning and renaming of columns.
    4. Generation of RDF using morph-KGC python library

  8. What can you do with this application?

    The following actions can be performed

    1. Search schools by name.
    2. Filter schools that are connected by a given Metro line.
    3. Filter schools that belong to a certain district.
    4. Filter schools by their accessibility level.
    5. Combine all these filters.
    6. Add schools you like into a favourite schools list and the access them later.
    7. Look for more information by clicking each card.
