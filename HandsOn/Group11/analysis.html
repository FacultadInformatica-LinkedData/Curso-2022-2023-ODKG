ANALYSIS

Data characteristics and schema

Field name CSV Field name(XML - Dublin Core Q) Description Comments
ID dc:identifier ID identifing each line in the dataset This column was created by us. We will use it to identify each plaque in the dataset. This ID has ben taken from the URL identifier. In this url there is one ID parameter.
Titulo dc:title Title on the plaque It is NOT unique. There are repeated values.
There are NO NULL values.
For example Benito Perez Galdos has 3 different plates in different locations.
Fecha dc:coverage.temporal Year of placement It is not unique. There are no null values. There are many plates placed in the same year.

Range: 1990 - 2022
Fecha de colocacion dc:coverage.temporal.inicio Date of placement. dd/mm/yyyy It is not unique.There are no null values.
There are many plates placed in the same year. There are 79 different values.

Range: 01/01/1990 - 09/03/2022
Lugar dc:coverage.spatial City It is unique for all records with the value Madrid since they are the commemorative plates just from Madrid.
Direccion dc:coverage.spatial Street where the plaque is located It is not unique. There are several plaques placed at the same address. There are no null values.
For example the plaque of Teatro Apolo and the plaque of Lope de Vega (both are at 45 Alcal√° street).
Distrito dc:coverage.spatial District There are no unique values. It is not null. There are 18 different values. The district with more plaques is Centro.

ERROR: There is an error in the data. We can see plaques in these different district Moncloa-Aravaca, Mocloa-Aravaca and Moncloa-Ravaca appear. The 3 refer to the same district but the have been written differntly. To fix it we should replace all 3 for the same value: Moncloa-Aravaca.
Latitud dc:coverage.spatial Latitude in an ETRS89 reference system. It is not unique. There are no null values.

Range: 4.38 - 40.47
Longitud dc:coverage.spatial Longitude in an ETRS89 reference system. It is not unique. There are no null values.

Range: (-3.75) - (-3.64)
Palabras clave dc:description Key words about the historical character mentioned in the plaque It is not unique. There is no null value. The most repeated keywords are writers and madrid plaques.
Comentario dc:description Text on the plaque It is unique. Each plate has its own text.
AportadoPor dc:contributor Service providing the data Its unique for all lineas in the dataset: www.memoriademadrid.es
URL dc:identifier Resource URL It is unique because it is the resource identifier. The url is differentiated by an ID of each resource.
The url is form a search engine to which some query parameters are passed.
Editado dc:publisher Resource Editor Unique value: Biblioteca Digital memoriademadrid (Ayuntamiento de Madrid)
Disponible dc:date.available Date of publication of the resource It is not unique. On 08/29/2013 they put most of the resources (302 lines registered that day).
From that date they have been updating the lines.
Type dc:type File type It is a single IMAGE value for all records. In the url of each record, the image of the plate appears.
ERROR: There is an error one of the lines that has the value TEXT. However, it should be an IMAGE because the format indicated in the next column indicates that it is an image.jpeg.
Format dc:format Mime types (standard way of sending data over the Internet) All lines have the same value: image.jpeg
Language dc:language Language All lines have the same value: spa (SPANISH)
Rights dc:rights Rights of use Same value for all records. Creative Commons
Rightsholder dc:rightsholder Rights management organization Same value for all records: Biblioteca Digital memoriademadrid
